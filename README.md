# cfcrudgen
ColdFusion crud generator for MSSQL, includes support for:
### Select
- Count
- Exists
- Select
- SelectAll
- SelectRecord
- Paginate

### Remove
- Delete
- DeleteAll
- Remove

### Manipulate
- Insert
- Update
- Upsert
- Merge
- Save

## To Install
Run the following from commandbox:
`box install cfcrudgen`
